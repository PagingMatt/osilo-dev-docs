Cannot_decrypt ciphertext nonce is raised when cannot decrypt ciphertext with nonce with the secret key provided to decrypt.
val encrypt : key:Cstruct.t -> plaintext:Cstruct.t -> Cstruct.t * Cstruct.t
encrypt ~key ~plaintext encrypts the plaintext using the secret key and returns a pair containing the ciphertext and the nonce generated to be used for the encryption.

decrypt ~key ~ciphertext ~nocne decrypts the ciphertext with the secret key, using nonce.
